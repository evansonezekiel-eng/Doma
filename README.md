# Doma's Cosy Autumn Quest

This folder is ready to host as a small web app/PWA.

Permanent hosting options:
1. Netlify: log in, drag this folder or ZIP into Netlify Drop, then click Claim site.
2. GitHub Pages: upload these files to a public repository, then enable Pages from the main branch.
3. Vercel: import the folder/repository and deploy.

On iPhone: open the public link in Safari, tap Share, then Add to Home Screen.
